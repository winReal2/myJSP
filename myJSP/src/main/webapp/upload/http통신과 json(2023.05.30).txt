json (파일형식)
예전에는 XML형식의 파일을 사용했으나 지금은 json많이 사용
(json 특징)
- 텍스트로만 구성 > 속도빠름
- 언어나 플랫폼에 독립적! 여러 언어에서 많이사용
- 자바오브젝트를 json 형식으로 변환하기 쉽다(반대도 가능)
   (java Object > json / json > java Object)


json에서는 속성에 반드시!  큰따옴표(" ")사용! (가장 큰 차이점)
속성(이름)에는 -(하이픈), 공백사용가능하나 잘 사용하지 않음
json의 값 : 숫자, 문자열, 불린, 널, 배열만 사용 (함수사용X)
